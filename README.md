# Portfolio

Personal developer portfolio built with React, Vite, and Tailwind CSS.

## Run it

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```

## Stack

React · Vite · Tailwind CSS · React Router · lucide-react · react-icons

## Edit content

Update files in `src/data/` to change what shows on the site.
